            Getting Started

            1. Browse — Select a blueprint from the sidebar to view its structure
            2. Visualize — FSM blueprints render as interactive state diagrams and procedure flow graphs
            3. Inspect — Click nodes to see procedure details, steps, standards references, and evidence
            4. Export — Download as YAML, JSON, or Mermaid diagram via the Export button

            Blueprint Types

            FSM Blueprints
            Finite State Machine definitions for audit methodologies (ISA, IIA-GIAS). Each procedure is an event-sourced aggregate with states, transitions, commands, and guards.
            Generation Profiles
            Complete DataSynth YAML configurations for specific use cases: fraud detection, ML training, process mining, audit engagement simulation.
            Scenarios
            Overlay configurations that inject specific conditions: supplier fraud schemes, trading anomalies, control failures, going concern triggers.
            Regional Templates
            Country-specific naming data (vendors, customers, materials, line items) for generating culturally authentic synthetic data.
            Causal DAGs
            Directed acyclic graphs modeling causal relationships between macro conditions, operational parameters, and financial outcomes.
